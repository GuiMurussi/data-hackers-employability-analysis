# 📊 Análise de Empregabilidade na Área de Dados (Data Hackers)

Projeto de análise de dados baseado na pesquisa **State of Data Brazil**, realizada pela comunidade Data Hackers.

O objetivo deste projeto é explorar dados sobre profissionais da área de dados no Brasil, analisando aspectos como diversidade, escolaridade, senioridade e distribuição geográfica.


## Objetivo

Investigar padrões de empregabilidade na área de dados no Brasil e apresentar os resultados por meio de **visualizações interativas**, facilitando a interpretação dos dados.


## Dataset

Os dados utilizados neste projeto são provenientes da pesquisa **State of Data Brazil**, que reuniu **4.273 respostas de profissionais da área de dados**.

A pesquisa analisa informações como:

- gênero
- etnia
- escolaridade
- nível de senioridade
- distribuição geográfica
- média salarial

## Principais análises realizadas:

### Diversidade na área de dados
- Distribuição por **gênero e etnia**
- Comparação entre diferentes grupos demográficos

### Escolaridade
- Relação entre **gênero e nível de formação**
- Distribuição entre graduação, pós-graduação, mestrado e doutorado

### Senioridade profissional
- Distribuição entre níveis:
  - Júnior
  - Pleno
  - Sênior
  - Gestão

### Distribuição geográfica
- Visualização da distribuição de profissionais da área de dados no Brasil

### Média salarial
- Comparação salarial entre diferentes grupos de gênero
